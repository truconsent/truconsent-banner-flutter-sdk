# TruConsent Flutter SDK Example App

Example application demonstrating the TruConsent Flutter SDK integration.

## Setup

1. Navigate to the example directory:
```bash
cd example
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

## Features

- Input fields for API credentials (API Key, Organization ID, Banner ID, User ID)
- Pre-filled default values for quick testing
- Button to open consent banner modal
- Display last consent action
- Test checklist UI
